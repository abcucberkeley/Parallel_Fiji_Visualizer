1. Copy the _lib folder to the lib\win64 folder in the root of your Fiji installion. Example: Fiji.app\lib\win64
2. Copy the _plugin folder to the plugins folder in the root of your Fiji installation. Example: Fiji.app\plugins
3. Now launch a new session of Fiji and you should be able to see the plugin under the Plugins tab! (You can also search for "Parallel Fiji Visualizer" in the search bar)
